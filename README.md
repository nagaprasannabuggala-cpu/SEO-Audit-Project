# SEO Audit Project – Snitch (Men’s Jeans Category)

##  Project Overview
This project focuses on conducting a complete SEO audit for Snitch.com, specifically the Men’s Jeans category. It includes analysis of technical SEO, on-page optimization, keyword research, and competitor insights to understand how SEO impacts website performance and visibility.

## Tools Used
- **Semrush** – SEO audit and keyword research  
- **Ubersuggest** – Keyword ideas and search volume analysis  
- **PageSpeed Insights** – Website speed and Core Web Vitals analysis  
- **Similarweb** – Traffic and competitor analysis  

## Objectives
- Analyze technical, on-page, and off-page SEO  
- Identify issues affecting search rankings  
- Perform keyword research based on search intent  
- Recommend improvements to increase organic traffic  

##  Key Activities

### 1. SEO Audit
- Conducted a full audit covering technical, on-page, and off-page SEO  
- Identified major issues in content, speed, and optimization  

### 2. Keyword Research
- Identified high-volume keywords for the men’s jeans category  
- Categorized keywords into **Transactional, Informational, and Navigational**  

### 3. On-Page Analysis
- Evaluated title tags, meta descriptions, and headings  
- Identified lack of content on product listing pages (PLP)  
- Suggested improvements for keyword usage  

### 3. Technical SEO Analysis
- Analyzed Core Web Vitals (LCP, CLS, INP)  
- Identified performance issues affecting user experience  
- Suggested improvements for mobile optimization  

### 4. Competitor Analysis
- Compared Snitch with competitors to identify SEO gaps  
- Analyzed keyword and content opportunities  

## ⚠️ Key Issues Identified
- Thin or missing content on category pages  
- Weak keyword optimization  
- Limited internal linking  
- Moderate domain authority and backlinks  

## 💡 Recommendations
- Add 200–300 words SEO content on category pages  
- Optimize title tags and meta descriptions  
- Improve internal linking  
- Build backlinks through content and outreach  
- Improve website speed and mobile performance  

## 📈 Expected Outcomes
- Increase in organic traffic  
- Improved keyword rankings  
- Better user engagement  
- Higher conversion potential  

## 📊 Learning Outcome
- Gained practical understanding of SEO fundamentals  
- Learned keyword research and search intent  
- Understood technical and on-page SEO concepts  
- Developed ability to identify and solve SEO issues
